# Politique de sécurité

Merci de contribuer à la sécurité de Dofus Calculator.

## Signaler une vulnérabilité

Merci de ne pas publier publiquement les détails d'une faille avant qu'un correctif soit disponible.

Pour un signalement :

- utilisez la fonctionnalité de signalement privé GitHub si elle est activée sur le dépôt
- sinon, contactez le mainteneur du dépôt via GitHub avec une description claire, les étapes de reproduction et l'impact estimé

## Ce qu'il faut inclure

- type de vulnérabilité
- étapes de reproduction
- impact potentiel
- version, commit ou environnement concerné
- proposition de correctif si vous en avez une

## Processus

Le projet essaiera de :

- accuser réception rapidement
- confirmer le problème ou demander des précisions
- préparer un correctif avant toute divulgation publique

Merci d'agir de bonne foi et de laisser le temps nécessaire à la correction.
